# open-api-docker

## Quickstart

```
docker-compose up -d
```

## Requirements

- swagger-editor
- swagger-ui
- prism
- redoc

## Url

| url | services |
| ---- | ---- |
| http://localhost:8001/ | swagger-editor |
| http://localhost:8002/ | swagger-ui |
| http://localhost:8003/users | mock-api (prism) |
| http://localhost:8004/ | redoc |
